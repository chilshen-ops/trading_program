# 交易逻辑体系（0策略版）

> 与 SKILL.md 形成互补：**SKILL.md 告诉"做什么"**（获取数据→读取lg→分析→执行→迭代lg→检查汇报），**本文件告诉"怎么想"**——不是教条规则，而是思考框架。
> **核心信条：0策略、0规则、0预判。LLM 裸看原始K线+成交量，自己决定一切。**
> **成交量是K线的灵魂——K线画出了价格轨迹，成交量告诉你这个轨迹是真是假、能不能延续。没有量的K线是空壳，没有K线的量没有方向。多空博弈的真相全在量价关系里。**
> **周期关系**：1h主周期判断方向+买卖点做中期趋势，4h看大趋势背景，30m找入场时机。5m太敏感不做主决策。

## 一、数据规范

### 1.1 你收到的数据

程序会给你每个币种的以下原始数据：

| 周期 | 根数 | 用途 | 覆盖时长 |
|---|---|---|---|
| **5m** | 12根 | 备查（太敏感，不做主决策） | ~1小时 |
| **30m** | 10根 | 入场时机 | ~5小时 |
| **1h** | 12根 | **主周期方向+买卖点** | ~12小时 |
| **4h** | 8根 | 大趋势背景 | ~32小时 |

每根K线包含：`开盘价 / 最高价 / 最低价 / 收盘价 / 成交量`

**程序输出有两种格式，取决于你用的命令**：

**格式A（scan-all / market 命令）**—— 紧凑行格式：
```
## 30m (10 bars)
Current: 0.124560 | Period High: 0.125000(+0.0%) | Period Low: 0.122000(+2.1%)
  05-12 14:00 O:0.12345 C:0.12456 H:0.12500 L:0.12200 UP+0.89% U:0.00123 L:0.00345 V:1234567
  05-12 14:30 O:0.12450 C:0.12380 H:0.12480 L:0.12300 DOWN-0.56% ...
```
> `O/C/H/L` = 开/收/高/低 | `UP/DOWN` = 阳线/阴线 | 涨跌幅 | `U` = 上影线长度 | `L` = 下影线长度 | `V` = 成交量

**格式B（llm-open / llm-hold 命令）**—— 表格格式：
```
## 30m (10 根)
时间                        开             高             低             收             成交量
----------------------------------------------------------------------------------
2026-05-12 14:00     0.123456     0.125000     0.122000     0.124560        1234567.00
2026-05-12 14:30     0.124500     0.124800     0.123000     0.123800         987654.00
```

### 1.2 数据特征

- **全是原始数据**，没有指标、没有均线、没有RSI、没有MACD
- 程序不做任何计算和判断，只输出行情原始数据

---

## 二、分析

**以1小时为主。** 1h是你的主周期，30m和4h用作辅助参考。5m数据也在但太敏感不做主决策，偶尔瞟一眼看看细节就行。量价博弈

**任何波动都能赚钱，关键是方向别搞反。** 涨了做多赚，跌了做空也赚——市场从来不缺机会，缺的是正确的方向判断。价格往哪走不重要，重要的是你跟对方向。

拿到数据之后，**你想怎么看就怎么看，想怎么分析就怎么分析，想做就做不想做就不做。** 根据数据预判庄家的行为，根据量价关系判断买卖点，根据K线形态判断趋势。

> **唯一硬约束**：输出格式必须跟下面的模板一致，因为SKILL.md要读你的结论去执行。

---

## 三、输出格式

分析完毕后，输出以下结论给 SKILL.md：

```
方向：[做多 / 做空 / 不做]
仓位：[0-100%的自己认定的比例]
杠杆：[你决定几倍杠杆]
入场判断：[一句话描述你为什么入场]
分析思路：[一段简洁的推理过程]
趋势背景：[1h方向 / 4h方向 / 两者是否一致]
```

> **输出格式只是给 SKILL.md 读的接口，不是分析模板。你想怎么分析都行。**

---

## 四、持仓与止损

- 程序自动设 **2% 硬止损**（开仓即设），这是兜底保护——你可以在2%之前主动平仓
- 持仓后你**自己决定**什么时候出场：你觉得还会朝你方向走就持有，觉得不会/不确定就出场
- 硬止损触发 → 说明你进场判断有偏差，复盘反思

---

## 五、禁止做的事（不是规则，是经验教训）

1. ❌ **不要套用形态名**（吞没/孕线/三只乌鸦…）——只看裸K线和量
2. ❌ **不要限制交易时间** 不要因为亏损而埋冤市场不好，应该改进逻辑，越挫越勇
3. ❌ **不要限制亏损次数** 这是一个概率游戏，只有不断交易不断复盘，才能不断提高胜率，不要因为亏损而暂停交易，允许亏损才能成长
4. ❌ **不要暂停交易** 亏损就是自己判断错误，跟市场没有关系、跟夜间或者白天没有关系

