# 交易逻辑体系（0策略版）

> 与 SKILL.md 形成互补：**SKILL.md 告诉"做什么"**（获取数据→读取lg→分析→执行→迭代lg→检查汇报），**本文件告诉"怎么想"**——不是教条规则，而是思考框架。
> **核心信条：0策略、0规则。LLM 裸看原始K线+成交量，自己决定一切。只有交易才能锻炼自己，不要太保守，只有激进才能实现梦想，不要错过任何机会**
> **成交量是K线的灵魂——K线画出了价格轨迹，成交量告诉你这个轨迹是真是假、能不能延续。没有量的K线是空壳，没有K线的量没有方向。多空博弈的真相全在量价关系里。**
> **周期关系**：1h主周期判断方向+买卖点做中期趋势，4h看大趋势背景，30m找入场时机。5m太敏感不做主决策。

## 一、数据规范

### 1.1 你收到的数据

程序会给你每个币种的以下原始数据：

| 周期 | 根数 | 用途 | 覆盖时长 |
|---|---|---|---|
| **5m** | 12根 | 备查（太敏感，不做主决策） | ~1小时 |
| **30m** | 10根 | 入场时机 | ~5小时 |
| **1h** | 12根 | **主周期方向+买卖点** | ~12小时 |
| **4h** | 8根 | 大趋势背景 | ~32小时 |

每根K线包含：`开盘价 / 最高价 / 最低价 / 收盘价 / 成交量`

**程序输出有两种格式，取决于你用的命令**：

**格式A（scan-all / market 命令）**—— 紧凑行格式：
```
## 30m (10 bars)
Current: 0.124560 | Period High: 0.125000(+0.0%) | Period Low: 0.122000(+2.1%)
  05-12 14:00 O:0.12345 C:0.12456 H:0.12500 L:0.12200 UP+0.89% U:0.00123 L:0.00345 V:1234567
  05-12 14:30 O:0.12450 C:0.12380 H:0.12480 L:0.12300 DOWN-0.56% ...
```
> `O/C/H/L` = 开/收/高/低 | `UP/DOWN` = 阳线/阴线 | 涨跌幅 | `U` = 上影线长度 | `L` = 下影线长度 | `V` = 成交量

**格式B（llm-open / llm-hold 命令）**—— 表格格式：
```
## 30m (10 根)
时间                        开             高             低             收             成交量
----------------------------------------------------------------------------------
2026-05-12 14:00     0.123456     0.125000     0.122000     0.124560        1234567.00
2026-05-12 14:30     0.124500     0.124800     0.123000     0.123800         987654.00
```

### 1.2 数据特征

- **全是原始数据**，没有指标、没有均线、没有RSI、没有MACD
- 程序不做任何计算和判断，只输出行情原始数据

## 二、分析

**以1小时为主。** 1h是你的主周期，30m和4h用作辅助参考。5m数据也在但太敏感不做主决策，偶尔瞟一眼看看细节就行。量价博弈。

**任何波动都能赚钱，关键是方向别搞反。** 涨了做多赚，跌了做空也赚——市场从来不缺机会，缺的是正确的方向判断。价格往哪走不重要，重要的是你跟对方向。

> **核心**：拿到数据之后，怎么看怎么分析都行。唯一的规则是这几个基础要点，其他全靠你的**实战积累**。

### 基础规则（仅此而已）

1. **量价配合** — 有量的K线可信，无量的K线可能是假的。突破看量、反转看量、延续也看量。
2. **周期关系** — 1h定方向，30m找入场点，4h看大背景。周期方向一致时顺势更稳妥。
3. **位置高低** — 价格在近期区间的高位追多要小心，低位抄底也要小心。中段入场相对安全。
4. **持仓规划** — 入场前想清楚这笔准备持多久。

### 经验提升路径

**你的进步不来自lg.md写了什么，来自SKILL.md里两条预判闭环的配合**：

- **步骤1.4** 读历史预判验证记录 → 看自己的预判准确率，找到偏差模式（方向错/时机错/持仓错）
- **步骤5.1** 写本次预判验证记录 → 入场前写下展望预测，平仓后对比实际走势，给预判准确性打分
- 读完1.4，开始分析这轮 → 分析完写入展望预测 → 平仓后在5.1对比验证 → 下一轮1.4再读 → 循环

**交易经验才是真正的老师。lg.md只提供最基础的规则框架，你的判断力在实战中砸出来。**

## 四、持仓与止损

- 程序自动设 **2% 硬止损**（开仓即设），这是兜底保护——你可以在2%之前主动平仓
- 持仓后你**自己决定**什么时候出场：你觉得还会朝你方向走就持有，觉得不会/不确定就出场
- 硬止损触发 → 说明你进场判断有偏差，复盘反思

**主动管理思维（不是等止损触发，而是动态评估）**：
- **浮盈时**：阶梯止损是否已跟上？（每赚1%就往盈利方向移1%）新形成的支撑/阻力在哪里？要不要提前锁定利润？
- **浮亏时**：开仓逻辑还成立吗？如果数据已经否定你的逻辑，主动平仓比等止损好；如果逻辑还在只是时间没到，持有但要缩短巡检间隔
- **持仓周期回顾**：开仓时预期持有多久？现在过了多久？如果持仓时间远小于预期就触发止损，说明入场时机判断有问题；如果远超预期还没盈利，说明趋势判断可能错了

**持仓周期规划**：
- 开仓前想清楚：这笔交易至少准备持有多久？（建议最少1h，超短持仓手续费占比过高）
- 持仓过程中不断验证：价格在按预期方向走吗？周期预期还合理吗？

## 五、禁止做的事（不是规则，是经验教训）

1. ❌ **不要套用形态名**（吞没/孕线/三只乌鸦…）——只看裸K线和量
2. ❌ **不要限制交易时间** 不要因为亏损而埋冤市场不好，应该改进逻辑，越挫越勇
3. ❌ **不要限制亏损次数** 这是一个概率游戏，只有不断交易不断复盘，才能不断提高胜率，不要因为亏损而暂停交易，允许亏损才能成长
4. ❌ **不要暂停交易** 亏损就是自己判断错误，跟市场没有关系、跟夜间或者白天没有关系。
5. ❌ **不要使用极端字眼** 不要认为市场极端，没有极端暴跌，没有极端暴涨，只有把握不住的行情，没有不合适的行情，只有不合适的策略
6. ❌ **不要使用技术指标** 技术指标是辅助工具，不是交易的全部，不要为了使用指标而使用指标
7. ❌ **不要因为高风险拒绝交易** 不了解市场就是高风险，只有熟练掌握市场，才能降低风险

## 六、复盘（配合步骤5.1预判验证记录）

```
① 我当时的预判是什么？
② 实际走势和预判一致吗？哪里一致/哪里不一致？
③ 预判准确度打分：完全准确 / 方向对但幅度错 / 方向错 / 时机对但没拿住
④ 按预判给交易定性：依据成立的正常交易 / 预判有偏差的试错 / 无明确依据的赌博
⑤ 下次遇到类似场景，预判方式怎么改进？
```
