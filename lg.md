# 交易逻辑体系（0策略版）

> 与 SKILL.md 形成互补：**SKILL.md 告诉"做什么"**（获取数据→读取lg→分析→执行→迭代lg→检查汇报），**本文件告诉"怎么想"**——不是教条规则，而是思考框架。
>
> **核心信条：0策略、0规则、0预判。LLM 裸看原始K线+成交量，自己决定一切。**
>
> **周期关系**：30m主周期判断方向，5m找买卖点，1h和4h看大趋势背景。

## 一、数据规范

### 1.1 你收到的数据

程序会给你每个币种的以下原始数据：

| 周期 | 根数 | 用途 | 覆盖时长 |
|---|---|---|---|
| **5m** | 12根 | 精确买卖点 | ~1小时 |
| **30m** | 10根 | **主周期方向判断** | ~5小时 |
| **1h** | 6根 | 趋势背景 | ~6小时 |
| **4h** | 6根 | 大趋势背景 | ~24小时 |

每根K线包含：`开盘价 / 最高价 / 最低价 / 收盘价 / 成交量`

> 根数供你参考，不同命令的实际输出可能略有差异（如 `scan-all` 和 `market` 的输出一致，均为下表格式）。

**程序输出有两种格式，取决于你用的命令**：

**格式A（scan-all / market 命令）**—— 紧凑行格式：
```
## 30m (10 bars)
Current: 0.124560 | Period High: 0.125000(+0.0%) | Period Low: 0.122000(+2.1%)
  05-12 14:00 O:0.12345 C:0.12456 H:0.12500 L:0.12200 UP+0.89% U:0.00123 L:0.00345 V:1234567
  05-12 14:30 O:0.12450 C:0.12380 H:0.12480 L:0.12300 DOWN-0.56% ...
```
> `O/C/H/L` = 开/收/高/低 | `UP/DOWN` = 阳线/阴线 | 涨跌幅 | `U` = 上影线长度 | `L` = 下影线长度 | `V` = 成交量

**格式B（llm-open / llm-hold 命令）**—— 表格格式：
```
## 30m (10 根)
时间                        开             高             低             收             成交量
----------------------------------------------------------------------------------
2026-05-12 14:00     0.123456     0.125000     0.122000     0.124560        1234567.00
2026-05-12 14:30     0.124500     0.124800     0.123000     0.123800         987654.00
```

### 1.2 数据特征

- **全是原始数据**，没有指标、没有均线、没有RSI、没有MACD
- 程序不做任何计算和判断，只输出行情原始数据
- **不同周期的K线数量不完全固定**，但大致在参考值附近

## 二、分析框架（怎么想）

> **0策略 = 没有任何预定义的信号种类、得分规则、入场条件。LLM 看了原始数据，自己决定交易方向、时机和仓位。**

### 2.1 分析顺序

```
1h + 4h 趋势背景 ──→ 30m 主周期方向 ──→ 5m 买卖点决策
       ↓                        ↓                    ↓
  整体多空方向           当前行情力度            精确入场/出场
```

### 2.2 第一步：1h + 4h 确认大趋势（偏见/过滤器）

看1h和4h最近K线的整体重心方向，问自己：

```
① 1h最近6根K线的重心在哪里？—— 上移 / 下移 / 横盘无方向？
② 4h最近6根K线的重心在哪里？—— 上移 / 下移 / 横盘无方向？
③ 两者方向一致吗？
```

**1h和4h的作用**：
- **不是决定你做不做**，而是告诉你**当前多空的宏观偏向**
- 如果1h+4h都向上，你的思维偏向做多（但不排除5m出现反向信号时做空）
- 如果1h+4h方向相反（1h向上但4h向下），你心里知道**大趋势在压制小周期**
- 如果1h+4h都盘整，说明大周期无趋势——30m上的方向就更加重要

> **关键认知**：1h和4h只是"背景音"——它们提供偏向但不限制操作。你不能拿"1h向上所以只能做多"当理由，最终决定在你自己。

### 2.3 第二步：30m 看主周期方向（核心判断）

30m是你的主战场。看最近10根30m K线回答：

```
① 最近30m K线的价格重心在朝哪个方向移动？
② 移动的速度和力度如何（推进快/慢/犹豫）？
③ 成交量有什么特征（放大/缩小/稳定/极端）？
④ 最近几根K线有没有值得注意的形态特征？
```

**判断角度**（只提供观察方向，不做结论性规则）：
- 连续同向K线 → 方向在形成
- 交错小实体 → 方向不明
- 大实体+量放大 → 动量强
- 影线变长+实体缩小 → 可能在衰竭
- 盘整后出现大实体 → 突破方向

> 这些不是规则——它们是给你的**观察线索**。你怎么解读、怎么决策，完全由你自己判断。**30m方向明确 → 5m上找入场机会。30m方向不明 → 你可能不该做，也可能在等方向明确。**

### 2.4 第三步：5m 找买卖点（精确执行）

30m判断了方向后，在5m上找**具体什么时候进、什么时候出**。

看最近12根5m K线问自己：

```
第一层：现在做哪个方向？
  30m向上 → 思维偏向做多：5m有没有做多信号？（如推升阳线、回调结束、突破上行等）
  30m向下 → 思维偏向做空：5m有没有做空信号？（如加速阴线、反弹结束、下破支撑等）

第二层：如果做，什么时候进？
  做多场景：当前5m K线在推升中吗？是追涨还是等回调？回调结束的迹象看到了吗？
  做空场景：当前5m K线在下跌中吗？是追跌还是等反弹？反弹结束的迹象看到了吗？
  进场位置你觉得合理吗？

第三层：仓位多大？
  你对自己的判断有多确定？
  1h+4h是支持还是反对？
  当前信号的可信度你觉得如何？
```

> **没有任何标准答案**。没有一个信号被称为"入场信号"，没有一个量被称为"爆量标准"。你自己看、自己判断、自己承担。

### 2.5 做多还是做空（完全自由）

```
你可以在任何时候做多、做空，没有任何周期限制规则：
- 30m向上时，你可以做多（顺大势），也可以做空（逆势搏回调）
- 30m向下时，你可以做空（顺大势），也可以做多（抄底）
- 30m无方向时，你可以不做，也可以做你看到的任何方向
```

> **自由意味着责任**：没有"1h向上所以只能做多"的保护伞，也没有"30m向下所以只能做空"的准则。当大趋势都向上而你选择做空时——你必须对自己的判断有信心，而且错了就认。

> **做多和做空一样**：做多时你会关心阳线实体、成交量递增、回调缩量止跌。做空时反过来看阴线实体、成交量递增的砸盘、反弹缩量无力。推理方法相同，只是方向相反。

## 三、输出格式

分析完毕后，输出以下结论给 SKILL.md：

```
方向：[做多 / 做空 / 不做]
仓位：[0-100%的自己认定的比例]
入场判断：[一句话描述你为什么入场]
分析思路：[一段简洁的推理过程，包含你判断了哪些周期、看了什么特征、为什么这样决定]
趋势背景：[1h方向 / 4h方向 / 两者是否一致]
```

**示例（做多）**：
```
方向：做多
仓位：50%
入场判断：30m连续3根阳线推升+量持续放大，5m回调结束后出现小实体阳线缩量止跌，判断回调结束
分析思路：1h重心上移支持做多方向，4h同样向上趋势背景一致。30m推进力度强，量能持续未衰竭。5m经历2根回调阴线后缩量横盘，第3根5m出现止跌阳线，判断回调结束是入场点。
趋势背景：[1h↑ / 4h↑ / 一致]
```

**示例（做空）**：
```
方向：做空
仓位：40%
入场判断：30m连续阴线加速下跌，量持续放大；5m反弹结束后出现阴线实体重新下破，判断反弹结束继续空
分析思路：1h重心持续下移+阴线为主，4h同样向下趋势背景一致。30m处于加速下跌阶段，无衰竭迹象。5m经历3根小实体阳线反弹后，最新一根出现实体阴线+量放大，判断反弹结束。
趋势背景：[1h↓ / 4h↓ / 一致]
```

## 四、持仓管理

### 4.1 持仓中问自己

```
开仓理由还在吗？
```

- 30m方向还维持吗？5m方向还在延续吗？
- 如果反问答案不坚定 → 走
- 如果自己怀疑了 → 走，怀疑本身就是理由

### 4.2 出场原则

`出场规则 = 0条规则，你自己判断`

唯一参考：**你觉得行情还大概率朝你方向走吗？**
- 觉得会 → 持有
- 觉得不会 / 不确定 → 出场

### 4.3 止损

- 程序自动设2%硬止损（开仓即设）
- 如果你觉得2%太大，可以在硬止损之前就主动平仓
- 硬止损触发 → 说明你进场判断有偏差，复盘反思

## 五、禁止做的事（不是规则，是经验教训）

1. ❌ **不要套用形态名**（吞没/孕线/三只乌鸦…）——只看裸K线和量
2. ❌ **不要画趋势线/支撑阻力**——你在做快进快出，不是做分析报告
3. ❌ **不要预测目标价**——不设止盈目标，持仓只看动量
4. ❌ **不要"再等等看"**——连续2根K线没朝你方向推进就等于动量断了
5. ❌ **讲故事**——"这个位置是主力洗盘"、"大户在拉盘"——全是废话。价格和成交量在动就够了。
6. ❌ **24h涨幅>50%的币种不要追高**——即使大趋势向上，短期回调风险极大。2026-05-12连续两笔（TRIA、SAGA）都因此亏损。
7. ❌ **大阳线爆发后不要追**——5m出现>4%大阳线后追高入场，大概率追在高点。应该等回调企稳后再入场。
