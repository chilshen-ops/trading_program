# lg.md — 交易逻辑手册

> **只分析 OHLCVA（价格 + 成交量），不依赖 MACD/RSI/均线**
> 价格和成交量是唯一真实的市场语言。指标是滞后的，K线不会骗人。

---

## 一、核心信念


1. **只分析量价** — 不依赖任何技术指标，只看价格（K线实体/影线）和成交量（量能大小/变化）
2. **方向优先于数值** — 方向对了，每根K线都赚钱；方向错了立即走，不硬扛
3. **量价整体** — 价格和成交量不可分，放量上涨 = 真实趋势，缩量上涨 = 诱多
4. **预判区间，不赌单点** — 看支撑到压力的空间，而非纠结精确价格
5. **持仓周期灵活** — 有利润持有，方向错立即走，不等回到开仓价

## 二、入场判断：量价结构信号

### 做空信号（暴涨后回落）

**核心：暴涨 → 诱多 → 量能衰竭 → 高点下移**

| 阶段 | 信号标准 |
|------|---------|
| 暴涨 | 1h 涨幅 ≥ 5%，量能 ≥ 前一小时的 80% |
| 诱多 | 冲高后长上影线（影线 ≥ 实体 3 倍），或连续两根阴线回落 |
| 量能衰竭 | 量能阶梯递减：连续 3 根 K 线量能低于前一小时的 50% |
| 高点下移 | 1h 高点持续下移（双高/三高下移确认） |

**做空优先级：**
- P0（最佳）：高点下移 + 量能递减
- P1：暴涨后长上影线 + 量能递减
- P2：暴涨后横盘 + 极度缩量（极度缩量 = 变盘前兆）

### 做多信号（瀑布后企稳）

**核心：暴跌 → 缩量 → 底部结构 → 阳线**

| 阶段 | 信号标准 |
|------|---------|
| 瀑布 | 1h 跌幅 ≥ 5%，量能放大 |
| 缩量 | 量能递减：连续 3 根 K 线量能低于瀑底那根的 50% |
| 底部结构 | 低点抬高，或连续 2 次测试同一支撑位不破 |
| 阳线出现 | 缩量后底部阳线（实体放大 + 缩量） |

**做多优先级：**
- P0（最佳）：低点抬高 + 缩量 + 阳线
- P1：暴跌后连续缩量 + 底部结构形成
- P2：极度缩量横盘后放量选择方向

### 禁止信号

- 暴涨币做多 / 暴跌币做空（逆势）
- 横盘 + 无量（方向不明，不做）
- 支撑位连续 3 次测试不破（底部结构形成中，不做空）
- 压力位连续 3 次测试不破（顶部结构形成中，不做多）

## 三、开仓标准

**必须同时满足才能开仓：**

```
[ ] 方向信号明确（做空信号 ≥ P1，或做多信号 ≥ P1）
[ ] 量能配合（放量变盘，不是无量横盘）
[ ] 止损空间合理（距止损位 ≥ 5%，防止被扫）
[ ] 非黑名单币种（已读取 blacklist.json）
[ ] 保证金 ≤ 总资产的 2%（模拟盘保守）
```

## 四、持仓管理

**每次巡检按顺序执行，每项必须给出结论：**

### 检查顺序

**1. 开仓理由还在吗？**
- 回顾开仓时的分析逻辑，新数据还支持吗？
- 结论：延续 / 动摇 / 反转

**2. 支撑/压力位是否被突破？**
- 支撑 2 次不破 = 底部预警 → 警惕
- 支撑 3 次不破 = 底部结构确认 → **立即平仓**
- 压力 2 次不破 = 顶部预警 → 警惕
- 压力 3 次不破 = 顶部结构确认 → **立即平仓**

**3. 止损位检查**
- 浮盈 ≥ 1% → 移保本（止损移至入场价）
- 浮盈 ≥ 3% → 逐步锁利（部分平仓或收紧止损）
- 浮盈从高点回吐 50% → 重新评估，必要时平仓

**4. 持仓时长**
- 超过 4 小时 = **红色警戒**，下一轮必须决策（平仓或持有）

**5. 量能异常**
- 突然放量 2 倍 + 方向不变 → 警惕（可能主力出货/进货）
- 极度缩量 + 价格横盘 → 变盘前兆，不提前平仓

### 持仓决策表

| 情况 | 决策 |
|------|------|
| 做空逻辑延续 + 浮盈安全 | 持有 |
| 底部结构形成（支撑 3 次不破）| **立即平仓** |
| 横盘超 4h | **立即平仓** |
| 浮亏 + 逻辑动摇 | **立即平仓** |
| 浮盈 ≥ 3% + 趋势延续 | 部分锁利或持有 |

## 五、平仓标准

| 平仓类型 | 条件 | 执行 |
|----------|------|------|
| 止损触发 | 价格触及止损位 | 市价全平 |
| 主动平仓（底部结构）| 支撑位 3 次不破 | 立即平仓 |
| 主动平仓（横盘）| 横盘超 4h | 立即平仓 |
| 主动平仓（逻辑动摇）| 开仓理由不成立 + 价格未到止损 | 立即平仓 |
| 止盈 | 价格达到预设止盈位 | 全平或部分锁利 |
| 浮盈回吐警戒 | 浮盈从高点回吐 50% | 重新评估，必要时平仓 |

**原则：执行纪律 > 方向判断。底部结构形成后，即使方向判断正确也要立即走。**

## 七、预判体系

> Kronos 核心贡献：量价整体 + 分层预测 + 无教师强制 + 零样本预测

### 7.1 量价整体（最重要）

- **放量上涨 = 真实趋势** → 持有或追入
- **缩量上涨 = 诱多** → 警惕，可能反转
- **放量下跌 = 真实抛压** → 做空逻辑支持
- **缩量下跌 = 可能见底** → 警惕，可能反转

### 7.2 区间预判（不看单点）

- 预判价格到**支撑区间**或**压力区间**的空间，而非赌单点价格
- 入场决策基于：区间内价格均值偏向哪边
- 方向判断优先于具体价格

### 7.3 极端事件预警（跳空 = 信号断裂）

- **跳空检测**：|open_t / close_{t-1} - 1| 超过阈值 → 结构性断裂，原有预判可能失效，需重新评估
- **量能突变**：突然放大 2 倍以上 → 主力介入，方向可能加速或反转
- **价格 + 成交量背离**：价格上涨但成交量未跟上 → 预警信号，动能衰竭
- **极度缩量横盘后放量**：变盘前兆，配合均线判断方向

### 7.4 不同周期不同参数

- **1h 找方向**：趋势清晰则跟踪，趋势模糊则观望
- **30m/15m 找切入点**：1h 方向明确后，等待回调后入场
- **多周期共振**：1h + 30m 方向一致 → 信号最强
